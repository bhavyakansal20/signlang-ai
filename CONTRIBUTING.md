# 🤝 Contributing Guidelines

Thank you for your interest in contributing 🚀
This project is maintained by **Bhavya Kansal** and focuses on real-world AI applications.

---

## 🧠 Contribution Scope

You can contribute by:

* Improving model performance or efficiency
* Enhancing UI/UX or deployment
* Fixing bugs or issues
* Improving documentation
* Suggesting better approaches

---

## 🛠 How to Contribute

1. Fork the repository
2. Create a new branch:

```bash
git checkout -b feature/your-feature-name
```

3. Make your changes
4. Commit your changes:

```bash
git commit -m "feat: improve model performance"
```

5. Push your branch:

```bash
git push origin feature/your-feature-name
```

6. Open a Pull Request (PR)

---

## ⚠️ Important Guidelines

* Do NOT upload sensitive or private datasets
* Ensure all data used is legally shareable
* Avoid exposing API keys or secrets
* Keep code clean, modular, and readable
* Add comments where necessary

---

## 🔬 AI/ML Specific Guidelines

* Clearly explain model changes
* Include evaluation metrics if modifying models
* Avoid unnecessary complexity
* Maintain reproducibility where possible

---

## 🔀 Pull Request Rules

* Keep PRs focused and minimal
* Reference related issues if any
* Ensure code runs without breaking existing functionality
