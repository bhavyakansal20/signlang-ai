# 🔐 Security Policy

## 📌 Overview

This project involves AI/ML systems and is provided for educational and demonstration purposes.

---

## 🚨 Reporting Security Issues

If you discover:

* Model vulnerabilities
* Data leakage risks
* Misuse possibilities
* Security flaws in deployment

Please report responsibly.

## 🛡 Scope

This repository does not handle:

* Sensitive personal user data (directly)
* Production-grade security systems

Security concerns mainly relate to:

* Model integrity
* Data usage
* System misuse
